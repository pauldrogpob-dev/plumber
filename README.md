# WJ McNabb Plumbing Website

The official responsive landing page for WJ McNabb Plumbing, Pittsburgh's trusted plumber for 50+ years. Built with modern, high-performance web technologies.

## Technology Stack
- **Frontend Framework:** React 18
- **Bundler & Build Tool:** Vite
- **Styling:** Tailwind CSS & Custom CSS Variables
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the local development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
